## How to Use

1. **Learn by Chapter**: Start with `chapter_01_Basics` and progress sequentially
2. **Run Examples**: Each `.js` file contains runnable examples. Execute with:
   ```bash
   node filename.js
   ```
3. **Reference Materials**: Keep the PDF and markdown shortcut references open while coding
4. **Experiment**: Modify the examples and observe the output to deepen understanding
