# Chapter 1: Basics

## 01_Basics.js

A simple starter file to verify JavaScript execution.

### Code

```javascript
console.log("Hello, World!")
```

### What It Does

Prints `Hello, World!` to the console.

### How to Run

```bash
node 01_Basics.js
```

### Expected Output

```
Hello, World!
```

### Purpose

- Confirms Node.js is installed and working correctly
- Validates your local JavaScript execution environment
- First step before diving into advanced topics
