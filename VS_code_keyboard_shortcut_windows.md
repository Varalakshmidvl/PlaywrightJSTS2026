## Custom Keybindings

You can customize any shortcut by:

1. Press `Ctrl + K Ctrl + S` to open Keyboard Shortcuts
2. Search for the command you want to change
3. Click the keybinding and press your new shortcut
4. Or edit `keybindings.json` directly (`Ctrl + Shift + P` -> `Open Keyboard Shortcuts JSON`)

---

*Tip: Press `Ctrl + K` then wait - VS Code will show all chords (two-key shortcuts) that start with `Ctrl + K`.*

*Tip: Press `Ctrl + Shift + P` to open the Command Palette and type any command or keyboard shortcut.*
