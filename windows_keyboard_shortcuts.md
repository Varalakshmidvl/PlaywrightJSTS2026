# Windows Keyboard Shortcuts - Complete Reference

## General System Shortcuts

| Shortcut | Action |
|----------|--------|
| `Win` | Open/Close Start Menu |
| `Win + D` | Show Desktop (minimize all windows) |
| `Win + M` | Minimize all windows |
| `Win + Shift + M` | Restore minimized windows |
| `Win + E` | Open File Explorer |
| `Win + R` | Open Run dialog |
| `Win + I` | Open Settings |
| `Win + X` | Open Quick Link menu (Power User) |
| `Win + L` | Lock PC |
| `Win + S` | Open Search |
| `Win + A` | Open Action Center/Notification Center |
| `Win + V` | Clipboard history |
| `Win + .` or `Win + ;` | Open emoji panel |
| `Win + PrtScn` | Screenshot and save to Pictures |
| `Win + Shift + S` | Snipping Tool / Screenshot |
| `Win + Tab` | Open Task View |
| `Win + Ctrl + D` | Add virtual desktop |
| `Win + Ctrl + F4` | Close virtual desktop |
| `Win + Ctrl + Left/Right` | Switch between virtual desktops |

## Window Management

| Shortcut | Action |
|----------|--------|
| `Alt + Tab` | Switch between open apps |
| `Alt + F4` | Close active window |
| `Alt + F4` (on desktop) | Shut down dialog |
| `Alt + Space` | Open window menu |
| `Win + Left/Right` | Snap window to left/right half |
| `Win + Up` | Maximize window |
| `Win + Down` | Minimize/restore window |
| `Win + Shift + Left/Right` | Move window to other monitor |
| `Win + Home` | Minimize all except active window |
| `Win + ,` (comma) | Peek at desktop |
| `F11` | Fullscreen mode (in most apps) |
| `Ctrl + Shift + Esc` | Open Task Manager directly |
